========================================
CARPETA DE FICHAS TÉCNICAS PDF
========================================

Esta carpeta contiene los archivos PDF de las fichas técnicas de los productos.

INSTRUCCIONES:
--------------

1. Coloca aquí los archivos PDF de cada producto
2. Usa el mismo nombre del producto como nombre del archivo
3. Ejemplo para "Logus GL3300AM":
   - Nombre del archivo: Logus_GL3300AM.pdf
   - Los espacios se reemplazan con guiones bajos (_)

4. El botón "Ver Ficha Técnica PDF" en el modal del producto
   abrirá automáticamente el PDF correspondiente.

ESTRUCTURA RECOMENDADA:
-----------------------
assets/pdfs/
├── generadores-nafta/
│   ├── Logus_GL3300AM.pdf
│   ├── Logus_GL3300E.pdf
│   ├── Logus_GE3300E_PREMIUM.pdf
│   └── ...
├── generadores-diesel/
│   └── ...
└── inverter/
    └── ...

NOTAS:
------
- Los archivos PDF deben estar en formato .pdf
- Recomendado: tamaño máximo 5MB por archivo
- Si el PDF no existe, el botón intentará abrirlo pero el navegador
  mostrará un error 404 (página no encontrada)
